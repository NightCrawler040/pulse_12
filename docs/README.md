## ⚡ Краткий обзор системы

**Pulse12 FlowSpace** — это специализированная корпоративная система, разработанная для внутреннего использования в компании. Ее главная цель — дать руководителю (начальнику) удобный инструмент для постановки, распределения и контроля сроков выполнения задач, а командам — прозрачное рабочее пространство с мгновенным обменом статусами и уведомлениями.

### 🌟 Ключевые преимущества
1. **Мгновенная синхронизация (WebSockets):** Любое перемещение карточки, комментарий или изменение статуса моментально отражается у всех онлайн-пользователей без перезагрузки страницы.
2. **Гибридная надежность (Dual-Mode DB):** Система работает на базе **PostgreSQL 16** в бинарном формате `JSONB`. При потере связи с сервером БД она автоматически бесшовно переключается в локальный файловый режим (`db.json`).
3. **Безопасность корпоративного уровня:** Ролевая модель (Admin, Manager, Member), автовыход через 15 минут неактивности, защита главного администратора от удаления и валидация дубликатов учетных записей.
4. **Интерактивность:** Burn-down таймер обратного отсчета спринта, чек-листы готовности команды (`🤝 Готовность к сдаче`) и VIP-уведомления при упоминании через `@Имя`.
